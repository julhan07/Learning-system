# Requirements

-   PHP Version 8^
-   Laravel 8
-   Install composer
-   Install Database MYSQL

-   ReactJs
-   React Router
-   Mysql DB
-   PHP Laravel Server

# Initialize project

-   clone https://github.com/julhan07/Learning-system
-   composer update
-   npm install / yarn

# How to run:

-   make run (run laravel server)
-   make dev-js (run javascrypt)
